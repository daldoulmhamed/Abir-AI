## ğŸ’¡ Top 10 Prompts to Get Started

### 1. Content Ideation
```
Generate 10 [content type] ideas about [topic] for [audience]. 
For each idea, include: title, key angle, and why it would resonate.
```

### 2. Email Writing
```
Write a professional email to [recipient] about [purpose]. 
Tone: [friendly/formal]. Length: [100-150 words]. 
Include [specific points to cover].
```

### 3. Data Analysis
```
Analyze this data and identify: 
1) Key trends, 2) Anomalies, 3) Actionable insights. 
Data: [paste or describe data]
Audience: [who needs to understand this]
```

### 4. Meeting Summary
```
Summarize these meeting notes into:
1) Executive summary (3 sentences)
2) Key decisions made
3) Action items with owners and deadlines
Notes: [paste notes]
```

### 5. Learning New Concepts
```
Explain [complex concept] to me like I'm [beginner/intermediate]. 
Use analogies and examples. Then give me 3 practice problems.
```

### 6. Code Generation
```
Write [language] code to [task description].
Requirements: [list specific requirements]
Include: comments, error handling, example usage
```

### 7. Debugging Help
```
This code produces [wrong output] instead of [expected output].
Code: [paste code]
Input: [test input]
Help me identify the issue and fix it.
```

### 8. Content Improvement
```
Improve this [content type] to be more [engaging/clear/professional]:
[paste content]

Focus on: [tone/structure/clarity]
Target audience: [description]
```

### 9. Spreadsheet Formulas
```
Create an Excel formula that [what you want to calculate].
Column A: [description]
Column B: [description]
Result should: [specific requirement]
```

### 10. Brainstorming
```
I need creative solutions for [problem/challenge].
Context: [background]
Constraints: [limitations]
Generate 10 unconventional ideas I might not have considered.
```
